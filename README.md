# Shipmates-Practice

A disposable repository for learning and testing the ShipMates development
workflow safely.

## Purpose

This repository is where we exercise:

- isolated task branches and worktrees;
- automated validation;
- draft pull requests and GitHub checks;
- human approval bound to an exact pull-request head commit;
- guarded merge and cleanup.

It must not contain production credentials or production data.

## Development

```sh
npm test
```
